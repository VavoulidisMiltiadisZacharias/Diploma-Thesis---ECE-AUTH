# Diploma-Thesis---ECE-AUTH
A python based implelentation for the identification of key genes in cancerous cells through a comparative study of dimensionality reduction techniques based on graph theory and machine learning.

### **General info:**
This project evaluates how different dimensionality reduction techniques can be used to identify and rank important genes in genomics data (specifically gene expression) for cancer research. The implementation is built entirely in Python. Feel free to download and edit the code.
- *Keywords*: Computational Oncology, Gene Expression, Dimensionality Reduction, Graph Theory, Machine Learning, Genes Ranking
<br/>

### **Setup:**
- Python 3.8 version or higher is recommended
- Jupyter Notebook required to run and view the `.ipynb` project files.
- Install all necessary data science and machine learning dependencies (pandas, numpy e.t.c.)
<br/>

### **How to run:**
First, set up your local repository and update the file paths used for loading and saving the dataset to match your local environment.
To replicate the analysis, please execute the files in the following order.
1. **`Project 4 - Data preprocessing.ipynb`**
   * *Purpose:* Cleans the raw microarray data and saves the processed dataset.
2. Remaining Notebooks.
   * *Note: Once the data preprocessing step is complete, the remaining notebooks can be executed in any order.

### **Complete stuff:**
A comparative analysis of 7 dimensionality reduction (DR) techniques algorithms using 4 machine learning (ML) algorithms in computational oncology context. The DR techniques and the ML algorithms used are the following:
- DR techinques:
  - Low Variance Filter
  - High Pearson Correlation Filter
  - High Spearman's Rank Correlation
  - Low Mutual Information Filter
  - Nearest Neighbor Graph (NNG)
  - Multiple Distance Graph (MDG)
  - Combined Distacne Graph (CDG)
- ML algorithms:
  - MultiLayer Perceptron
  - k-th Nearest Neighbor
  - Adapptive Boosting
  - Naive Bayes
